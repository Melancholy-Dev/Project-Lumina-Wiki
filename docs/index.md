# **Project Lumina Official Wiki**

Project Lumina is a narrative universe set in a futuristic era where magic and technology coexist. Within this universe, characters and events come to life in stories, games, and more.

The universe and its related contents are mainly created and published by Melancholy and Coetus Interactive. However, Project Lumina is open to everyone. Anyone, simply by giving credit to the project and its creators, can make fan-made content and release it as part of the Project Lumina universe.

To suggest changes to this wiki or contribute with new material, open a pull request on GitHub.

- [Contributors](contributors.md): the people currently credited for the project.
- [Wiki layout guide](wiki-layout.md): a reference for writing future pages in a consistent wiki format.

## Project status

Several Project Lumina products are in development, including a tabletop role-playing game, video games, this wiki, and a manga series. Some projects are active, while others are paused until the setting, resources, or team capacity are ready.
