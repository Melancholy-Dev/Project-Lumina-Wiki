# **Wiki Layout Guide**

Each main page should follow this order:

1. **Title:** use `# **Title**` heading with the page name.
2. **Short summary:** one or two sentences explaining what the page is about.
3. **Quick facts:** type, status, creator, first appearance, or other key details.
4. **Main sections:** split long text into clear `##` sections.
5. **Related pages:** links to products, characters, locations, or contributors.
6. **Missing information:** use an admonition when the page is incomplete.

## Product page template

```md
# Product Name

One or two sentences describing the product.

## Quick facts

- **Type:** game, manga, manual, story, or other format
- **Status:** planned, in development, paused, released, or archived
- **Creator:** name or team
- **Related pages:** links to relevant pages

## Overview

Short explanation of the product and its place in Project Lumina.

## Development status

Current state, known release plans, and what is still missing.

## Links

- [Official page](page.md)

!!! note "Missing information"

    List the details that still need to be added.
```

## Writing rules

- Prefer short paragraphs over long blocks of text.
- Use links with descriptive text instead of bare URLs.
- Mark incomplete pages as stubs instead of leaving them empty.
